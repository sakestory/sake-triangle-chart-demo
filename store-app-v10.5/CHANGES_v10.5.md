# 店内版 診断アプリ 改修メモ（v10.4 → v10.5）

対象: SAKE story 店内版「日本酒三角チャート診断」`index.html`
元ソース: `Netlify公開用_v10.4_2.zip` 内の index.html（本物の v10.4）

## 変更点

1. **占い版ロジックの撤去**
   - 全国市販酒の三選リスト `const RECS3 = {…}` を削除。
   - 結果画面の「あなたにおすすめの銘柄三選！」見出し・◇◎★の入手マーク凡例・「今夜の一本」ブロック・`tonightLine()` 関数・`threeSel`/`tonight` 描画を削除。
   - 未使用になった旧CSS（`.three-item` / `.t-tier` / `.tonight` / `.legend-note` / `.famous-*`）も除去。
   - `recToday` / `recFamous` / `famous` フラグを廃止。
   - grep確認: `RECS3=` / 三選 / ◇◎★ / market / tier の残骸 **0件**（`RECS3` の語はコード内説明コメント1箇所のみ）。

2. **店の19銘柄を埋め込み（`STORE_SAKE`）**
   - 元栓 `銘柄座標マスター_v2.2.json` の `in_store:true`（19件）と **名前・座標が完全一致**（検証済み）。
   - 各銘柄に `comment:""`（橋野さん本人コメント欄）を用意。

3. **近い順ランキング（上位5本）**
   - ユーザー座標と各銘柄の3次元ユークリッド距離を計算し、近い順に上位5本を結果画面「あなたに近い、店の一杯」に表示。
   - 三角チャートのドット／ラベルも `STORE_SAKE` を参照（表示とランキングが一致）。

4. **自動理由文（仮）＋コメント差し替え構造**
   - `reasonFor()`：`comment` があればそれを最優先表示、無ければユーザーと銘柄の「共通軸」から橋野文体で自動生成。
   - 将来、19銘柄の `comment` に一言ずつ書き加えれば、そのまま置き換わる。

5. **バージョン表記** トップ・結果の2箇所を v10.4 → v10.5 に更新。
